# Basic review example

This directory shows a small, representative slice of the local workspace that `revpack prepare` creates for an agent review.

Included here:

- `CONTEXT.md` - a shortened example of the generated entry point
- `outputs/new-findings.json` - an example positional finding
- `outputs/replies.json` - an example reply to an existing thread
- `outputs/summary.md` - an example PR/MR summary
- `outputs/review.md` - an example optional review-level note

A real bundle also includes the full diff artifacts, thread files, schemas, generated instructions, and machine-readable state described in the main README.
