# Agent Output Files Are Lazy Drafts

Revpack treats files under `.revpack/outputs/` as agent-created draft review material, not prepare-time placeholders. Prepare writes revpack-owned schema references under the read-only `.revpack/schemas/` directory, but does not create empty `replies.json`, `new-findings.json`, `summary.md`, or `note.md`; missing conditional output files intentionally mean there is no draft material to publish. This keeps tool-owned bundle inputs outside the agent's writable output boundary and favors a clearer agent writing contract over a predictable but confusing set of empty placeholder files.
